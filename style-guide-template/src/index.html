Style Guide 1.0

Purpose/Theme

The theme chosen as inspiration to create the wireframe to style the Zen Garden webpage is called by the number 217 (http://csszengarden.com/217/) The original style (The Beauty of CSS Design) has some different colors and is organized in squares. Each square contains one text or section; They are presented in different sizes, depending on the amount of information it holds. The wireframes created, and presented below, use different colors; the main shapes are circles and squares with rounded corners. The texts are not shown at the same time, but each at a time when the user chooses it pressing the round button with each title.
